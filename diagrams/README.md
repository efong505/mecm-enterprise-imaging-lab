# Diagrams

## Purpose

This folder stores diagrams for the MECM Enterprise Imaging Lab.

Diagrams should help explain topology, role placement, content flow, deployment flow, and troubleshooting paths.

## Suggested Diagrams

- Lab topology diagram
- MECM role relationship diagram
- Imaging content flow diagram
- PXE boot flow diagram
- Task sequence flow diagram
- Client management flow diagram

## Diagram Standards

Each diagram should include:

- Title
- Date
- Related phase
- Scope
- Legend or notes
- Clear indication of whether the diagram is planned, draft, or implemented

## Boundaries

Do not include private infrastructure details, production IP addresses, real hostnames, credentials, or sensitive environment information.

Do not add fake screenshots or diagrams that imply completed work before the lab step has been performed.

## Notes

Add diagram source files or exported images later as the lab design develops.
